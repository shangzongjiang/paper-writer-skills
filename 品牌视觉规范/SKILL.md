# Anthropic 品牌样式

## 概述

使用此技能可获取 Anthropic 官方品牌标识和样式资源。

**关键词**：品牌推广、企业形象、视觉标识、后期处理、样式设计、品牌色彩、字体排版、Anthropic 品牌、视觉格式化、视觉设计

## 品牌规范

### 色彩

**主色调：**

- 深色：`#141413` - 主要文字及深色背景
- 浅色：`#faf9f5` - 浅色背景及深色上的文字
- 中灰色：`#b0aea5` - 次要元素
- 浅灰色：`#e8e6dc` - 微妙背景

**强调色：**

- 橙色：`#d97757` - 主要强调色
- 蓝色：`#6a9bcc` - 次要强调色
- 绿色：`#788c5d` - 第三强调色

### 字体排版

- **标题**：Poppins（Arial 备用）
- **正文**：Lora（Georgia 备用）
- **注意**：为获得最佳效果，请在您的环境中预先安装字体

## 功能特性

### 智能字体应用

- 将 Poppins 字体应用于标题（24pt 及以上）
- 将 Lora 字体应用于正文
- 在自定义字体不可用时自动回退到 Arial/Georgia
- 在所有系统上保持可读性

### 文字样式

- 标题（24pt+）：Poppins 字体
- 正文：Lora 字体
- 根据背景智能选择颜色
- 保留文字层级和格式

### 形状与强调色

- 非文字形状使用强调色
- 在橙色、蓝色和绿色强调色之间循环
- 保持视觉吸引力的同时符合品牌规范

## 技术细节

### 字体管理

- 可用时优先使用系统已安装的 Poppins 和 Lora 字体
- 提供自动回退到 Arial（标题）和 Georgia（正文）
- 无需安装字体——与现有系统字体兼容
- 为获得最佳效果，请在您的环境中预先安装 Poppins 和 Lora 字体

### 色彩应用

- 使用 RGB 色值实现精确的品牌色彩匹配
- 通过 python-pptx 的 RGBColor 类应用
- 在不同系统间保持色彩一致性
