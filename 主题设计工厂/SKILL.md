# Theme Factory 技能

本技能提供一套精心策划的专业字体与配色主题集合，每个主题均包含精心选配的色彩方案和字体搭配。选定主题后，即可将其应用于任意产物。

## 用途

若需为演示文稿幻灯片组应用一致的专业风格，请使用本技能。每个主题包含：
- 附有十六进制色值的完整配色方案
- 适用于标题和正文的互补字体搭配
- 适合不同场景与受众的独特视觉风格

## 使用说明

如需为幻灯片组或其他产物应用样式：

1. **展示主题展示文件**：显示 `theme-showcase.pdf` 文件，让用户直观查看所有可用主题。请勿对其进行任何修改，仅供预览使用。
2. **询问用户选择**：询问用户希望将哪个主题应用于幻灯片组
3. **等待确认**：获得用户对所选主题的明确确认
4. **应用主题**：主题确定后，将所选主题的配色和字体应用于幻灯片组/产物

## 主题详情

每个主题均在 `themes/` 目录中定义，包含完整规格说明：
- 附有十六进制色值的完整配色方案
- 适用于标题和正文的互补字体搭配
- 适合不同场景与受众的独特视觉风格

## 应用流程

选定首选主题后：
1. 从 `themes/` 目录读取对应的主题文件
2. 在幻灯片组中统一应用指定的配色和字体
3. 确保适当的对比度和可读性
4. 在所有幻灯片中保持主题的视觉一致性

## 创建自定义主题
若现有主题均不适合某个产物，可创建自定义主题。根据所提供的输入内容，生成一个与上述主题风格相似的新主题，并为其起一个能够体现字体/配色组合特点的类似名称。根据所提供的基本描述选择合适的配色和字体。生成主题后，先展示供审阅确认，再按上述方式应用该主题。
